# Customer_shopping_Analysis
Project on Customer shopping Analysis using Python, SQL and Power BI

Project Overview This project explores customer shopping  using transactional data from 3,900 purhcases across multiple product categories. The goal is to uncover key insights into spending habits, customer segments, product preferences, and subscription to support strategic business decisions.

**Key Objectives**


Load and explore the dataset using Python libraries like Pandas and Numpy
Perfrom data cleaning and feature engineering
Run SQL queries on the cleaned data for deeper business insights
Visualize insights using Power BI and Presentation






**Tools & Technologies**


Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL, MySQL
Power BI
Microsoft PowerPoint






**Business Recommendations**


Encourage subscription with exclusive benefits.
Launch loyalty programs for frequent buyers.
Optimize discount policies for balance sales and profits.
Focus marketing campaigns on high revenue age groups.
Promote top-rated and best selling products prominently.
